## Getting Started

### Prerequisites

- Node.js (v18 or later)
- PostgreSQL
- Redis
- Docker (optional)

### Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/outfit-matcher.git
cd outfit-matcher
```

2. Install dependencies:

```bash
npm install
```

3. Set up environment variables:

```bash
cp .env.example .env
```

4. Start the development servers:

```bash
npm start
```
